Table Notes

This table contains the following cases in which no SCOTUS precedent was cited at all. Therefore, there are no citation networks available for those cases.

  • Duncan v. Owens. In this federal habeas case, the writ of certoriari was dismissed as improvidently granted.
  • Montana v. Wyoming. In this original jurisdiction case regarding State water rights, the Court issued a one-page Order and Judgment without reference to any case law
  • Hawkins v. Community Bank. In this Equal Credit Opportunity Act case, the Court was equally divided for the first time since Justice Scalia's death. Judgment below was affirmed in a one-line opinion.
  • Friedrichs v. CA Teachers Assn. In this First Amendment case, the Court was equally divided and so Judgment below was affirmed in a one-line opinion. Abooda> was therefore left intact
  • Dollar General v. Mississippi Band of Choctow Indians. In this Tribal Jurisdiction case, the Court was equally divided and so Judgment below was affirmed in a one-line opinion.
  • United States v. Texas. In this case concerning President Obama's DAPA program, the Court was equally divided and so Judgment below was affirmed in a one-line opinion.

Network Terminology

  • Supreme Court citation networks show the citation connections between two Court cases. Every network connects two cases, the 2015 Term Case and the Network Anchor Case.
  • The Network Age expresses the amount of time (in years) between the time the Anchor Case was decided and when the 2015 Term Case was decided.
  • The "D" in "3D" and "2D" stands for "degree of connection." Since the 2015 Term Case always cites the Network Anchor Case, those cases are 1-degree connected.
  • The 2D cases column gives the number of 2-degree connections that exist between the 2015 Term Case and the Network Anchor Case - that is, how many cases exist in the network that are cited by the 2015 Term Case AND in turn cite the Network Anchor. For example, Case X would be a 2D case if the 2015 Term Case cited Case X and Case X in turn cited the Network Anchor Case. Please note that the 2D case number includes the two 1D cases (2015 Term Case + Network Anchor).
  • The 3D cases column gives the number of 3-degree connections that exist between the 2015 Term Case and the Network Anchor Case. 3D cases are those that are cited by a 2D case AND in turn cite the Network Anchor. Thus, Case Y would be a 3D case if Case X above Cited Y, which in turn cited the Network Anchor Case. Please note that the 3D case number includes the all the 2D+1D cases.
  • DOD stands for "Degree of Dissent". The DOD of an individual case is calculated based on its vote for outcome, as specified by the Supreme Court Database (Spaeth). Since a 5-4 decision is the maximum amount of dissent possible in a SCOTUS case, it has a DOD of 1.0. Conversely, a 9-0 case has a DOD of 0. This means that an 8-1 decision has a 0.25 DOD, a 7-2 has a 0.50 DOD, and a 6-3 has a 0.75 DOD. To calculate DOD, the number of dissents is multiplied by 0.25 (thus a 6-2 case would still have a 0.50 DOD).
  • 2D DOD is the average DOD for all the 2D cases.
  • 3D DOD is the average DOD for all the 3D cases.
  • tb stands for "too big" and appears when a 3D network contains more than 70 cases. See Method below for further explanation.

Method

Each network was created by the researcher using the free tool on CourtListener. Choice of the Network Anchor Case was based on a reading of the opinion and SCOTUSBlog's Every. The textual justification for the Network Anchor Case choice is provided in the visualization's "Description" field. Once an Anchor Case is chosen, the citation networks are generated automatically by the software.

Limitations. Spaeth codes are not yet available for 2015 Term cases - they should become available in July or August 2016. All maps will be updated then. Please note that the Degree of Dissent (DOD) calculations do not include the 2015 Term case. In addition, the tool only recognizes pre-2015 Term cases for which there is a Supreme Court Database (Spaeth) ID number. Since Spaeth does not cover pre-1946 Term cases, the tool cannot create networks to cases decided before 1946. In addition, the Supreme Court Database has not yet coded current Term cases and so these are displayed in Spaeth view as "unk" (unknown). Finally,since citations between cases are all parsed by computer, the tool does experience occasional data errors.
